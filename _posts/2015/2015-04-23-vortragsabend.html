Facebooks JavaScript-Bibliothek React (JS) und React Native und ihr virtueller DOM sorgen gerade für gehörigen Wirbel in der Frontendszene. Erst dieses Jahr wurde wurde React Native angekündigt, welches Entwicklern erlaubt mit JavaScript native Views zu rendern. Höchste Zeit für eine Einführung in React (JS), die Konzepte dahinter und mit Beispielen auch in die Möglichkeiten von React Native zu schauen.

Paul Wittmann ist JavaScript-, Ruby- und Clojure-Entwickler bei Railslove. Auf der Suche nach einer besseren Frontendarchitektur fand er React, was schnell zur Einstiegsdroge zu funktionaler Programmierung und ClojureScript wurde.

Philipp arbeitet als Frontend Entwickler ebenfalls bei Railslove. Er beschäftigt sich seit dem erscheinen von React Native mit dieser neuen Technik und ist bereits jetzt ein Fan. Mithilfe von Beispielen wird er die aktuellen Möglichkeiten beleuchten.

Anschließend mögen wir mit euch gerne über die Vor- und Nachteile, Einsatzzwecke und Probleme von Web-Apps und Hybrid-Apps diskutieren.
